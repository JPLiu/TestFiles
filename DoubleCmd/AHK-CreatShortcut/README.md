# CreatShortcut

用 AutoHotkey 写的小工具，用于在 Double Commander 中创建 LNK 快捷方式。

Double Commander 工具栏按钮配置如下：

- Command: AutoHotKeyU32.exe 或者 AutoHotKeyU64.exe

|               | Parameters                          |
|---------------|-------------------------------------|
| CreatShortcut | `AHK-CreatShortcut.ahk %LUs %"0%Dt` |
