# 自用广告过滤规则

## More-Rule

### Content-Farms-Lite.list

基于终结内容农场的规则转置的拦截规则。

### More-Rule\AdGuard-Cookies-Lite.list

基于 AdGuard Cookies 规则转置的精简规则，仅包含通用规则和通用排除规则，不含 if 类规则和站点特定规则。

## Liu's List

### Liu's-Block-List.list

不推荐订阅的个人自用拦截规则，规则来源为个人收集，涉及个人偏好，部分购物站点和资源隐藏的站点会被列入。

### Liu's-Clear-List.list

网址清理规则，作为 uBlock Origin 内置的 AdGuard URL Tracking Protection 规则的补充。

### Liu's-Static-List.list

不推荐订阅的个人用常规规则，内部规则可能涉及个人偏好随意移除页面栏目。

## Liu's-Style-List.list

不推荐订阅的个人用 Sytle 规则，内部部分规则涉及使用设备（如显示器）的分辨率等，不具备通用性。
