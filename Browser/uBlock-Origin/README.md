# 自用广告过滤规则

## More-Rule

### Content-Farms-Lite

基于终结内容农场的规则转置的拦截规则。

### Content-Farms-Lite-Hide

基于终结内容农场的规则转置的特化规则，用于在 Google 搜索结果隐藏对应栏目。

### AdGuard-Cookies-Lite

基于 AdGuard Cookies 规则转置的精简规则，仅包含通用规则和通用排除规则，不含 if 类规则和站点特定规则。

## Liu's List

### Liu's-Block-List

个人自用拦截规则，不推荐订阅，规则来源为个人收集，涉及个人偏好，部分购物站点和资源隐藏的站点会被列入

### Liu's-Clear-List

网址清理规则，作为 AdGuard URL Tracking Protection 规则的补充。

### Liu's-Static-List

个人自用常规规则，内部规则可能涉及个人偏好随意移除页面栏目。

### Liu's-Static-Desktop-Ext-List

个人自用常规规则，内部规则可能涉及个人偏好随意移除页面栏目，此为 Liu's-Static-List.list 的补充规则，包含仅限桌面端，不应在移动端生效的规则。

### Liu's-Style-List

个人自用 Sytle 规则，不推荐订阅，内部部分规则涉及使用设备（如显示器）的分辨率等，不具备通用性。

### Liu's-Vivaldi-List

个人自用规则，内容为 Vivaldi 内置广告过滤功能不支持，需要使用 AdGuard 订阅补充的规则。
