# WinOCRAHK

调用 IrfanView 进行截图，并在截图完成后使用 Windows.Media.Ocr.Cli.exe 对截图进行 OCR 的 AutoHotkey 脚本。

仅支持 Win 10 以上（Windows.Media.Ocr.Cli.exe 需要）。
